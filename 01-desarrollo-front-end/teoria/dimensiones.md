# Dimensiones

## Ancho y alto

- `width` para ancho

- `height` para alto


![image](https://github.com/eugenia1984/desarrollo-front-end-html-css-javascript/assets/72580574/ce3ec9fb-2329-4536-aabe-0fac4c1addd2)

```CSS
    div {
      background-color: dodgerblue;
      margin: 5px;
    }
    .a {
      width: 100px;
      height: 100px;
    }
    .b {
      width: 500px;
      height: 250px;
    }
    .c {
      width: 800px;
      height: 600px;
    }
  </style>
```
